# 📉 E-Commerce Break-Even ROAS Calculator

A lightweight, vanilla JavaScript calculator built for performance marketers and e-commerce founders to calculate true profitability on ad spend.

Most ad platforms optimize for Gross ROAS, ignoring hard costs. This tool factors in Cost of Goods Sold (COGS), shipping/fulfillment, and payment gateway fees to reveal your actual Break-Even ROAS and allowable Target CPA.

### 🚀 Live Demo & Web Tool
**Use the live interactive calculator here:** 👉 [Break-Even ROAS Calculator by David Tamachi](https://davidtamachi.ca/break-even-roas-calculator.html)

### Features
* **Real-Time Margin Calculation:** Instant feedback on gross vs. net margins.
* **Gateway Fee Factoring:** Automatically calculates standard Shopify/Stripe transaction fees (2.9% + $0.30).
* **No Server-Side Tracking:** 100% client-side JavaScript. No data is stored or sent to a database.

### Why This Was Built
In my work scaling e-commerce pipelines, I found that relying on in-platform metrics like "Cost Per Lead" or "Platform ROAS" without factoring in COGS leads to scaling at a loss. This tool was built to bridge the gap between media buying and business finance.
